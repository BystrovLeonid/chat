# Chat

A simple chat.

# Quick start
* Clone this repo
```
git clone https://github.com/bystrovleonid/chat.git
cd chat
```
* Install dependencies
```
npm install
```
* Start the React
```
npm start
```
* Start the socket server
```
node Server.js
```
* Open browsers and point them `http://localhost:3000`.
* Enjoy!
